## User Authentication

- 우리 커뮤니티에 회원가입 할 때, 사용자의 대표 서버와 캐릭터명을 입력하도록 한다.
- 바람의 나라 공식 사이트에서, 대표 캐릭터의 한줄 인사말에 커뮤니티에 가입한 ID를 입력하도록 한다.
- [한줄 인사말 예시](https://baram.nexon.com/Profile/Info?character=호불팔%40연)
- UTF-8 인코딩
- 테스트 캐릭터 정보 
- 대표 캐릭터 : 협가검@하자 
- 한 줄 인사말 : 승옹
